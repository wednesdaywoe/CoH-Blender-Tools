Welcome — first-run setup

The workshop needs to know where your City of Heroes install is. Point it at the folder that contains the Client directory (or the Client directory itself).

This is saved to workshop/config.json next to the launcher.

Files in working folder

Files extracted via the Browse tab land here. Edit them in Blender (open via File → Import → CoH Geometry in Blender, save the changes by re-exporting back over the same file), then click Stage to round-trip through the addon and drop a patch pigg into your install. Untick Round-trip if you want to package the file as-is without going through Blender.

Workshop-staged overrides

Patch piggs the workshop has placed in your install's patch directory. Removing one restores the original asset. Manually-placed piggs are not shown here — only ones whose filename starts with coh_workshop_.
